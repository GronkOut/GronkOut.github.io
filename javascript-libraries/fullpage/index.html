    Javascript - FullPage.js v3.1.2

    https://github.com/alvarotrigo/fullPage.js/tree/master https://alvarotrigo.com/fullPage/extensions

    Setup

    Use NPM

    Use html

    HTML structure

    section

    slide

    HTML properties

    fp-noscroll

    섹션에 스크롤바를 표시하지 않는다.

    fp-auto-height

    특정 섹션의 높이값을 자동으로 변경한다.

    fp-auto-height-responsive

    특정 섹션의 높이값을 자동으로 변경한다.
    반응형 동작을 추가한다.

    data-src

    미디어 파일을 지연 로드 (lazy loading) 한다.

    data-autoplay

    미디어를 섹션 진입 시 재생 (data-autoplay) 하고, 섹션 진출 시 정지 (data-keepplaying) 한다.

    Options

    licenseKey

    상업용 라이센스키를 입력한다.

    controlArrows

    키보드 화살표 키를 통해서 제어 가능 여부를 정한다.

    verticalCentered

    섹션 내 컨텐츠를 수직 센터 정렬한다.

    scrollingSpeed

    스크롤 애니메이션 속도를 밀리세컨으로 정한다.

    sectionsColor

    섹션의 배경색상을 설정한다.

    anchors

    각 섹션의 앵커를 설정한다.
    data-anchor 속성을 통해서 설정하는 것이 직관적이다.

    lockAnchors

    URL 의 앵커가 라이브러리에서 효과를 지닐지 말지를 결정한다.
    나만의 기능과 콜백을 위해 여전히 내부에서 앵커를 쓸 수는 있지만 웹사이트 스크롤에는 아무런 효과가 없다.
    URL 의 앵커를 사용하여 다른 플러그인과 결합 할 때 유용하다.

    easing

    https://github.com/alvarotrigo/fullPage.js/blob/master/vendors/easings.min.js

    vendors/easings.min.js 파일이 필요하다.

    easingcss3

    css3: true 일 경우 사용 할 효과를 정의한다.

    loopTop

    첫번째 섹션에서 스크롤을 위로 올릴 때, 마지막 섹션으로 이동 할 지 여부를 결정한다.

    loopBottom

    마지막 섹션에서 스크롤을 아래로 내릴릴 때, 첫번째 섹션으로 이동 할 지 여부를 결정한다.

    loopHorizontal

    슬라이더 루핑 여부를 결정한다.
    맨 처음 또는 맨 마지막으로 돌아가는 애니메이션을 재생한다.

    css3

    CSS 를 이용하여 애니메이션을 할 지 여부를 결정한다.

    autoScrolling

    automatic 스크롤을 쓸지 normal 스크롤을 쓸지 여부를 결정한다.

    fitToSection

    섹션을 화면 높이에 맞춘다.

    fitToSectionDelay

    섹션을 화면 높이에 맞출 때 지연 시간을 준다.

    scrollBar

    브라우저 스크롤바를 사용 할 지 여부를 결정한다.

    paddingTop

    섹션 상단의 여백을 정한다.

    paddingBottom

    섹션 하단의 여백을 정한다.

    fixedElements

    고정된 요소를 지정한다.

    normalScrollElements

    일반 스크롤 요소를 지정한다.
    overflow: scroll 또는 overflow-y: scroll 속성이 필요하다.

    bigSectionsDestination

    화면보다 큰 섹션으로 갈 때 스크롤 위치를 정한다.

    keyboardScrolling

    키보드를 사용하여 컨텐츠를 이동 할 지 여부를 결정한다.

    touchSensitivity

    스와이프가 동작하게 되는 거리 (민감도) 를 정한다.

    continuousVertical

    처음 또는 마지막 섹션에서 루핑 여부를 결정한다.
    맨 처음 또는 맨 마지막으로 가는 애니메이션이 보이므로 자연스럽지 못하다.

    continuousHorizontal

    https://alvarotrigo.com/fullPage/extensions/continuous-horizontal.html

    슬라이드 루핑 시점에서 자연스러운 이동을 한다.
    맨 처음 또는 맨 마지막으로 가는 애니메이션 없이 자연스럽다.
    유로 확장 플러그인을 구매해야 한다.

    scrollHorizontally

    https://alvarotrigo.com/fullPage/extensions/scroll-horizontally.html

    스크롤에 따라 섹션과 슬라이드 모두 동작한다.
    유로 확장 플러그인을 구매해야 한다.

    interlockedSlides

    https://alvarotrigo.com/fullPage/extensions/interlocked-slides.html

    모든 슬라이드의 활성 슬라이드 인덱스를 동기화 할 수 있다.
    유로 확장 플러그인을 구매해야 한다.

    dragAndMove

    https://alvarotrigo.com/fullPage/extensions/drag-and-move.html

    드래그를 사용하여 섹션 및 슬라이드를 이동 할 수 있다.
    유로 확장 플러그인을 구매해야 한다.

    offsetSections

    https://alvarotrigo.com/fullPage/extensions/offset-sections.html

    섹션을 전체화면이 아닌 구역으로 퍼센트에 기반하여 쓸 수 있는 방법을 지원한다.
    유로 확장 플러그인을 구매해야 한다.

    resetSliders

    https://alvarotrigo.com/fullPage/extensions/reset-sliders.html

    섹션 이동 후 슬라이더 활성 인덱스를 초기화한다.
    유로 확장 플러그인을 구매해야 한다.

    fadingEffect

    https://alvarotrigo.com/fullPage/extensions/fading-effect.html

    섹션 이동 시 페이드인 효과를 적용한다.
    유로 확장 플러그인을 구매해야 한다.

    animateAnchor

    URL 에 앵커가 있을 경우 바로 화면으로 이동 할 지 애니메이션으로 이동 할 지 여부를 결정한다.

    recordHistory

    섹션과 슬라이드 이동 시 브라우저 히스토리에 저장한다.
    autoScrolling: false 일 경우 사용 불가하다.

    menu

    메뉴 요소를 정한다.

    navigation

    섹션 인디케이터를 표시한다.

    navigationPosition

    섹션 인디케이터 표시 위치를 정한다.

    navigationTooltips

    섹션 인디케이터 툴팁을 지정한다.
    data-tooltip 속성을 통해서 설정하는 것이 직관적이다.

    showActiveTooltip

    섹션 인디케이터 툴팁 노출 여부를 결정한다.

    slidesNavigation

    슬라이더 인디케이터 표시 여부를 결정한다.

    slidesNavPosition

    슬라이더 인디케이터 표시 위치를 정한다.

    scrollOverflow

    https://github.com/alvarotrigo/fullPage.js/blob/master/vendors/scrolloverflow.min.js

    섹션의 높이보다 컨텐츠의 높이가 클 경우 스크롤 여부를 결정한다.
    vendors/scrolloverflow.min.js 파일이 필요하다.

    scrollOverflowReset

    https://alvarotrigo.com/fullPage/extensions/scrolloverflow-reset.html

    오버플로우 스크롤된 섹션에 재 진입 할 때 스크롤을 초기화한다.
    유로 확장 플러그인을 구매해야 한다.

    scrollOverflowOptions

    https://github.com/cubiq/iscroll

    iScroll 옵션을 지정 할 수 있다.

    sectionSelector

    섹션 클래스 선택자를 정 할 수 있다.

    slideSelector

    슬라이드 클래스 선택자를 정 할 수 있다.

    responsiveWidth

    지정된 값에 따라 브라우저 너비가 변하면 일반 웹사이트 스크롤 구조로 변경한다.

    responsiveHeight

    지정된 값에 따라 브라우저 높이가 변하면 일반 웹사이트 스크롤 구조로 변경한다.

    responsiveSlides

    https://alvarotrigo.com/fullPage/extensions/responsive-slides.html

    responsiveWidth 또는 responsiveHeight 설정에 따라 슬라이드를 섹션으로 변경한다.
    유로 확장 플러그인을 구매해야 한다.

    parallax

    https://alvarotrigo.com/fullPage/extensions/parallax.html

    배경 이미지에 패럴렉스 스크롤링 효과를 사용 할 수 있다.
    유로 확장 플러그인을 구매해야 한다.

    parallaxOptions

    parallax: true 일 경우 옵션을 설정한다.

    dropEffect

    https://alvarotrigo.com/fullPage/extensions/drop-effect.html

    섹션 이동 시 트랜지션 효과를 준다.
    유로 확장 플러그인을 구매해야 한다.

    dropEffectOptions

    dropEffect: true 일 경우 옵션을 설정한다.

    waterEffect

    https://alvarotrigo.com/fullPage/extensions/water-effect.html

    섹션 이동 시 트랜지션 효과를 준다.
    유로 확장 플러그인을 구매해야 한다.

    waterEffectOptions

    waterEffect: true 일 경우 옵션을 설정한다.

    cards

    https://alvarotrigo.com/fullPage/extensions/cards.html

    섹션 이동 시 트랜지션 효과를 준다.
    유로 확장 플러그인을 구매해야 한다.

    cardsOptions

    cards: true 일 경우 옵션을 설정한다.

    lazyLoading

    data-src 속성이 있는 모든 요소에 지연 로딩을 사용한다.
