    auto 브라우저의 기본동작에 맡기는 방식
    block FOIT 방식 (타임아웃까지 텍스트를 보여주지 않음)

    optional 100ms 내외의 시간 동안만 블럭하고 기본폰트 노출 (그 후에는 대체를 하지 않음)
