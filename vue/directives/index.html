    v-on

    https://v3.ko.vuejs.org/guide/events.html
    stop event.stopPropagation() 메서드를 호출
    prevent event.preventDefault() 메서드를 호출
    capture 이벤트 리스너에 캡쳐모드를 추가
    self 해당 엘리먼트에서 이벤트가 전달된 경우에만 동작
    { keyAlias } 특정 키에 대해서만 동작

    middle 마우스 가운데 버튼 이벤트에만 핸들러 동작

    Component two-way binding

    컴포넌트를 호출할 때 v-model:속성 으로 props 를 지정하고
    컴포넌트에서는 update:속성 으로 emit 하면 주석 처리된 부분을 생략할 수 있다.
    여러개의 모델을 바인딩하려면 v-mode:속성 으로 사용할 수 있지만
    1 개의 모델만 바인딩 할 때는 modelValue 로 속성 이름을 사용하는 것이 규칙이다.
