    Methods

    src

    https://gulpjs.com/docs/en/api/src

    파일 시스템에서 Vinyl 객체를 읽기 위한 스트림을 만든다.

    dest

    https://gulpjs.com/docs/en/api/dest

    Vinyl 객체를 파일 시스템에 쓰기 위한 스트림을 만든다.

    symlink

    https://gulpjs.com/docs/en/api/symlink

    Vinyl 객체를 파일 시스템에 연결하기 위한 스트림을 만든다.

    lastRun

    https://gulpjs.com/docs/en/api/lastrun

    현재 실행중인 프로세스중에 작업이 마지막으로 완료된 시간을 검색한다.
    src() 와 같이 사용하면 변경된 파일만 가져오기 때문에 성능에 유리하다.

    series

    https://gulpjs.com/docs/en/api/series

    태스크를 순차적으로 실행한다.

    parallel

    https://gulpjs.com/docs/en/api/parallel

    태스크를 동시에 실행한다.

    watch

    https://gulpjs.com/docs/en/api/watch

    glob 을 관찰하고 변경이 발생할 때 작업을 실행한다.

    task

    https://gulpjs.com/docs/en/api/task

    태스크를 정의한다.

    registry

    https://gulpjs.com/docs/en/api/registry

    사용자지정 레지스트리를 작업시스템에 연결하여 공유작업 또는 기능강화를 제공한다.

    tree

    https://gulpjs.com/docs/en/api/tree

    현재 작업 종속성 트리를 가져온다.
