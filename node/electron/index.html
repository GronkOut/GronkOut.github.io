    With vue on vite

    API

    Main process modules

    app

    https://www.electronjs.org/docs/latest/api/app

    앱의 이벤트 수명 주기를 제어한다.

    autoUpdater

    https://www.electronjs.org/docs/latest/api/auto-updater

    앱이 자동으로 업데이트되도록 한다.

    BrowserView

    https://www.electronjs.org/docs/latest/api/browser-view

    브라우저 창에 추가 웹 콘텐츠를 포함할 수 있다.

    BrowserWindow

    https://www.electronjs.org/docs/latest/api/browser-window

    브라우저 창을 만들고 제어한다.

    clipboard

    https://www.electronjs.org/docs/latest/api/clipboard

    시스템 클립보드에서 복사 및 붙여넣기 작업을 수행한다.

    contentTracing

    https://www.electronjs.org/docs/latest/api/content-tracing

    성능 병목 현상과 느린 작업을 찾기 위해 Chromium 에서 추적 데이터를 수집합니다.

    crashReporter

    https://www.electronjs.org/docs/latest/api/crash-reporter

    충돌 보고서를 원격 서버에 제출한다.

    desktopCapturer

    https://www.electronjs.org/docs/latest/api/desktop-capturer

    navigator.mediaDevices.getUserMediaAPI 를 사용하여 데스크톱에서 오디오 및 비디오를 캡처하는 데 사용할 수 있는 미디어 소스에 대한 정보에 액세스한다.

    dialog

    https://www.electronjs.org/docs/latest/api/dialog

    파일 열기 및 저장, 경고 등을 위한 기본 시스템 대화 상자를 표시한다.

    globalShortcut

    https://www.electronjs.org/docs/latest/api/global-shortcut

    앱의 키보드 포커스가 없을 때 키보드 이벤트를 감지한다.

    inAppPurchase

    https://www.electronjs.org/docs/latest/api/in-app-purchase

    Mac App Store 인앱 구매를 설정한다.

    ipcMain

    https://www.electronjs.org/docs/latest/api/ipc-main

    기본 프로세스에서 렌더러 프로세스로 비동기 통신한다.

    Menu

    https://www.electronjs.org/docs/latest/api/menu

    기본 앱 메뉴 및 상황에 맞는 메뉴를 만든다.

    MessageChannelMain

    https://www.electronjs.org/docs/latest/api/message-channel-main

    DOM MessageChannel 객체의 메인 프로세스 측에 해당한다.
    MessagePortMain 객체 쌍을 만든다.

    MessagePortMain

    https://www.electronjs.org/docs/latest/api/message-port-main

    DOM MessagePort 객체의 메인 프로세스 측에 해당한다.
    DOM EventTarget 시스템 대신 Node.js EventEmitter 이벤트 시스템을 사용한다는 점을 제외하고 DOM 버전과 유사하게 동작한다.

    nativeImage

    https://www.electronjs.org/docs/latest/api/native-image

    PNG 또는 JPG 파일을 사용하여 트레이, 도크 및 앱 아이콘을 만든다.

    nativeTheme

    https://www.electronjs.org/docs/latest/api/native-theme

    Chromium 기본 색상 테마의 변경 사항을 읽고 응답한다.

    net

    https://www.electronjs.org/docs/latest/api/net

    Chromium 의 기본 네트워킹 라이브러리를 사용하여 HTTP/HTTPS 요청 발행한다.

    netLog

    https://www.electronjs.org/docs/latest/api/net-log

    세션에 대한 네트워크 이벤트를 로깅한다.

    Notification

    https://www.electronjs.org/docs/latest/api/notification

    OS 데스크톱 알림을 만든다.

    parentPort

    https://www.electronjs.org/docs/latest/api/parent-port

    상위 프로세스와의 통신을 위한 인터페이스이다.

    powerMonitor

    https://www.electronjs.org/docs/latest/api/power-monitor

    전원 상태 변경을 모니터링한다.

    powerSaveBlocker

    https://www.electronjs.org/docs/latest/api/power-save-blocker

    시스템이 저전력 (절전) 모드로 전환되지 않도록 차단한다.

    process

    https://www.electronjs.org/docs/latest/api/process

    개체를 처리하기 위한 확장이다.

    protocol

    https://www.electronjs.org/docs/latest/api/protocol

    사용자 지정 프로토콜을 등록하고 기존 프로토콜 요청을 가로챈다.

    pushNotifications

    https://www.electronjs.org/docs/latest/api/push-notifications

    원격 푸시 알림 서비스에 등록하고 알림을 받는다.

    safeStorage

    https://www.electronjs.org/docs/latest/api/safe-storage

    로컬 시스템에 저장하기 위한 문자열의 간단한 암호화 및 암호 해독에 대한 액세스를 허용한다.

    screen

    https://www.electronjs.org/docs/latest/api/screen

    화면 크기, 디스플레이, 커서 위치 등에 대한 정보를 검색한다.

    session

    https://www.electronjs.org/docs/latest/api/session

    브라우저 세션, 쿠키, 캐시, 프록시 설정 등을 관리한다.

    ShareMenu

    https://www.electronjs.org/docs/latest/api/share-menu

    macOS 에서 현재 컨텍스트의 정보를 앱, 소셜 미디어 계정 및 기타 서비스와 공유하는 데 사용할 수 있는 공유 메뉴를 만든다.

    shell

    https://www.electronjs.org/docs/latest/api/shell

    기본 앱을 사용하여 파일 및 URL 을 관리한다.

    systemPreferences

    https://www.electronjs.org/docs/latest/api/system-preferences

    시스템 환경 설정을 가져온다.

    TouchBar

    https://www.electronjs.org/docs/latest/api/touch-bar

    네이티브 macOS 앱 용 TouchBar 레이아웃을 생성한다.

    Tray

    https://www.electronjs.org/docs/latest/api/tray

    시스템의 알림 영역에 아이콘과 상황에 맞는 메뉴를 추가한다.

    utilityProcess

    https://www.electronjs.org/docs/latest/api/utility-process

    Node.js 및 메시지 포트가 활성화 된 하위 프로세스를 생성한다.

    webContents

    https://www.electronjs.org/docs/latest/api/web-contents

    웹 페이지를 렌더링하고 제어한다.

    webFrameMain

    https://www.electronjs.org/docs/latest/api/web-frame-main

    웹 페이지 및 iframe 을 제어한다.

    Renderer Process Modules

    clipboard

    https://www.electronjs.org/docs/latest/api/clipboard

    시스템 클립보드에서 복사 및 붙여넣기 작업을 수행한다.

    contextBridge

    https://www.electronjs.org/docs/latest/api/context-bridge

    격리된 컨텍스트 간에 안전한 양방향 동기 브리지를 생성한다.

    crashReporter

    https://www.electronjs.org/docs/latest/api/crash-reporter

    충돌 보고서를 원격 서버에 제출한다.

    ipcRenderer

    https://www.electronjs.org/docs/latest/api/ipc-renderer

    렌더러 프로세스에서 메인 프로세스로 비동기식 통신한다.

    nativeImage

    https://www.electronjs.org/docs/latest/api/native-image

    PNG 또는 JPG 파일을 사용하여 트레이, 도크 및 앱 아이콘을 만든다.

    webFrame

    https://www.electronjs.org/docs/latest/api/web-frame

    현재 웹 페이지의 렌더링 사용자를 지정한다.
